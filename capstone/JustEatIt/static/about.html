About

Stock Price Plotter web app built using Flask, WT-Forms, Bootstrap, Quandl, Bokeh and Pandas under Python 2.7.

  • Flask - Web framework
  • WTForms - Web forms
  • Bootstrap - Elegant CSS
  • Quandl - Financial data API
  • Bokeh - Interactive plotting
  • Pandas - Dataframes
